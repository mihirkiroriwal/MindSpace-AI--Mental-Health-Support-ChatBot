# Mental Health AI Chatbot

A minimal AI chatbot designed to provide supportive conversations
for people experiencing stress, anxiety, or low mood.

## Features

- Local AI using Ollama
- Hinglish / Hindi / English support
- Crisis detection system
- Minimal chat UI
- Supportive long responses

## Tech Stack

- Python
- Streamlit
- Ollama
- Mistral / Llama3

## Disclaimer

This project is not a replacement for professional mental health care.
If you are experiencing a crisis, please contact a qualified professional
or a local helpline.
